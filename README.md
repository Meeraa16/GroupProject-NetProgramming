## Ansible Configuration

The Ansible environment was prepared to connect to a Cisco IOS router and execute network automation playbooks.

### Files Added

- `ansible.cfg` – Defines the inventory file and Ansible connection settings.
- `inventory.ini` – Contains the router host and management IP address.
- `group_vars/routers.yml` – Contains Cisco IOS connection variables.
- `playbooks/test_connection.yml` – Tests the Ansible connection to the router.
